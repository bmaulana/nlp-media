1) install Anaconda (https://www.anaconda.com/download/)
2) conda update conda
3) conda create -n nlp-media2 anaconda
4) activate nlp-media2
5) conda install -n nlp-media2 spacy
6) python -m spacy download en_core_web_lg
7) pip install -U cyhunspell
8) pip install -U vaderSentiment
9) pip install -U sentiment_classifier
10) Download files from https://github.com/kevincobain2000/sentiment_classifier/tree/master/src/senti_classifier
and copy to C:\\Users\\user\\Anaconda3\\envs\\nlp-media2\\lib\\site-packages\\senti_classifier\\
11) pip install --ignore-installed --upgrade tensorflow
12) Download CoreNLP: https://stanfordnlp.github.io/CoreNLP/ and unzip the folder to this directory's root
13) pip install -U stanfordcorenlp
14) pip install -U textblob
15) python -m textblob.download_corpora
